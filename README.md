## 🌟 About the Project

> Turning ideas into funded startups 🚀

**FoundIn** is a full-stack web platform designed to connect startup founders, investors, and innovators in one place.

It enables users to pitch ideas, discover funding opportunities, build networks, and gain insights through a modern and intuitive interface.

---

## 🎯 Core Features

- 🔐 Secure Authentication (JWT-based)  
- 📊 Personalized Dashboard  
- 💡 Startup Pitch Management  
- 💰 Funding Discovery & Tracking  
- 🌐 Networking System  
- 📈 Analytics & Insights  
- 👤 Profile Management  

---

## 🎨 UI Highlights

- Modern dark-themed UI  
- Clean and professional design  
- Fully responsive layout

## ⚙️ Tech Stack

**Frontend:** React.js, Tailwind CSS, Axios  
**Backend:** Node.js, Express.js  
**Database:** MongoDB, Mongoose  
**Authentication:** JWT

## 🚀 Getting Started

### Clone Repository
```bash
git clone https://github.com/your-username/foundin.git
cd foundin
```

### Backend Setup
```bash
cd backend
npm install
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in backend:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```
